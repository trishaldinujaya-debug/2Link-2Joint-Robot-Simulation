# 2-Link 2-Joint Robot Simulation 🤖

This project demonstrates the simulation of a 2-link, 2-joint robotic manipulator using MATLAB Simulink and Simscape Multibody. The system combines control theory with physical modeling to achieve realistic robotic motion.

## 🔧 System Overview

The simulation consists of:

* Reference Signal (Signal Builder)
* PID Controller (2DOF)
* Motor Model (Voltage → Torque)
* Encoder Feedback System
* 2-Link Mechanical Manipulator (3D Model)
* Scope for Output Visualization

## ⚙️ Features

* Real-time 3D robot movement
* Closed-loop PID control system
* Motor dynamics with feedback (Back EMF concept)
* Position tracking using encoder feedback
* Graphical analysis using Scope

## 🧠 Technologies Used

* MATLAB
* Simulink
* Simscape Multibody

## ▶️ How to Run

1. Open MATLAB
2. Run the `.m` file to initialize parameters
3. Open the `.slx` file in Simulink
4. Click **Run** to start simulation
5. View results in Scope and 3D Mechanics Explorer

## 📸 Output

* Robot motion in 3D
* Joint position vs time graphs
